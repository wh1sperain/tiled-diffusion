## 六、拼接质量评价标准

| 等级 | 分数 | 描述 |
| ---- | ---- | ---- |
| 优   | 5    | 拼接处完全无法察觉，纹理/结构自然连续 |
| 良   | 4    | 仔细观察可发现轻微差异，整体效果良好 |
| 中   | 3    | 拼接处有可见不连续，但不影响整体观感 |
| 差   | 2    | 拼接处明显不连续，纹理/颜色存在突变 |
| 极差 | 1    | 拼接处严重断裂，完全无法使用 |

## 七、模型选择说明

经评估，**本项目实验统一使用 SD 1.5**，不使用 SDXL，原因如下：

1. SDXL 管线（`sdxl/pipeline.py`）仅实现了简单的 X 轴边缘包裹（`wrap_edges_x`），未使用 `LatentHandler`
2. SDXL 不支持 `side_id`/`side_dir` 多 tile 连接系统，无法测试 one-to-one 和 many-to-many 模式
3. SDXL 不支持 `max_replica_width` 参数，无法做 similarity 约束消融
4. SDXL 无法进行 `LatentHandler.tile()` 中 padding 融合策略的改进实验
5. SD 1.5 的 512×512 分辨率在消融实验中显存开销更可控，实验效率更高
