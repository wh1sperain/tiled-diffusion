# 前置知识

## Variational Inference 变分推理

1. 变分图模型：

   假设从一张图 x 中存在隐变量 z 表征了图的特征

2. 变分推理：

   尝试用一种常见分布（正态分布...）来逼近未知分布的过程

3. 流程：

   - 假设 z 服从标准正态分布  -> z 的先验
   - 假设 后验分布 p(z|x) 服从 N(m, n) -> q(z|x)
   - 变分推理 -> 优化目标 = q(z|x) -> N(0,1)(先验)

1. 初始化带 padding 的 latent

2. graph_groups的结构

   ```python
   #单图 self-tiling
   side_id  = [1, 1, 2, 2]
   side_dir = ['cw', 'ccw', 'cw', 'ccw']
   
   graph_groups = {
       "0_0": {   # lat0 的 Right
           "target_latent_idx": [0],
           "target_side_idx":   [1]   # 匹配 lat0 的 Left
       },
       "0_1": {   # lat0 的 Left
           "target_latent_idx": [0],
           "target_side_idx":   [0]   # 匹配 lat0 的 Right
       },
       "0_2": {   # lat0 的 Up
           "target_latent_idx": [0],
           "target_side_idx":   [3]   # 匹配 lat0 的 Down
       },
       "0_3": {   # lat0 的 Down
           "target_latent_idx": [0],
           "target_side_idx":   [2]   # 匹配 lat0 的 Up
       }
   }
   
   #多图 many-to-many
   lat0:
   side_id  = [1, 1, None, None]
   side_dir = ['cw', 'ccw', None, None]
   
   lat1:
   side_id  = [1, 1, None, None]
   side_dir = ['cw', 'ccw', None, None]
   
   graph_groups = {
       "0_0": {   # lat0.Right
           "target_latent_idx": [0, 1],
           "target_side_idx":   [1, 1]
       },
       "1_0": {   # lat1.Right
           "target_latent_idx": [0, 1],
           "target_side_idx":   [1, 1]
       },
       "0_1": {   # lat0.Left
           "target_latent_idx": [0, 1],
           "target_side_idx":   [0, 0]
       },
       "1_1": {   # lat1.Left
           "target_latent_idx": [0, 1],
           "target_side_idx":   [0, 0]
       }
   }
   ```

   

3. Encoder -- AutoencoderKL

   - 目的：将原始图像压缩到 latent space
   - 方法：使用卷积神经网络进行降维，最后输出分布的参数，用于变分推理

   ```python
   输入图像 x: [B, 3, 512, 512]
           │
           ▼
   Conv 3x3
   [B, 128, 512, 512]
           │
           ▼
   Down Block 1 (ResNet/Conv, stride=2)
   [B, 128, 256, 256]
           │
           ▼
   Down Block 2
   [B, 256, 128, 128]
           │
           ▼
   Down Block 3
   [B, 512,  64,  64]
           │
           ▼
   Mid Block (ResNet + Attention)
   [B, 512,  64,  64]
           │
           ▼
   Norm + SiLU + Conv
   [B, 8, 64, 64]   ← 这里是 2 * latent_channels (latent_channels=4)
   
   Encoder 输出 h: [B, 8, 64, 64]
           │
           ▼
   quant_conv (1x1 Conv)
   [B, 8, 64, 64]
           │
           ▼
   按通道一分为二
   ├─ μ       : [B, 4, 64, 64]
   └─ logvar  : [B, 4, 64, 64]
           │
           ▼
   σ = exp(0.5 * logvar)
   ε ~ N(0, I)
   z = μ + σ ⊙ ε
           │
           ▼
   latent z: [B, 4, 64, 64]
   ```

   

4. Similarity Constraint

   - 目的：解决同组边各自演化导致的不一致（每个latent独自进行U-Net降噪）
   - 方法：对各组边，每次选取一个源边，将其内侧指定宽度（max_replica_width）写入各边内侧（padding与原latent范围为边界）；源边每次从组内按序选取，避免单一源边导致结构固化

5. Tiling

   - 目的：获取邻边的信息从而实现无缝生成
   - 方法：对于每个source边，根据graph_groups选择轮换候选边，对latent外沿进行padding
